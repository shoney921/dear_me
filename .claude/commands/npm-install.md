# NPM Install

프론트엔드 패키지를 설치합니다.

## 실행할 작업

frontend 디렉토리에서 npm install을 실행하여 의존성 패키지를 설치합니다.

## 명령어

```bash
cd /Volumes/shoney_SSD/dev/01_dear_me/frontend && npm install
```

## 설치 후 확인

타입 체크:

```bash
cd /Volumes/shoney_SSD/dev/01_dear_me/frontend && npm run typecheck
```

개발 서버 실행 (Docker 없이):

```bash
cd /Volumes/shoney_SSD/dev/01_dear_me/frontend && npm run dev
```

# DearMe - 수익화 로드맵

> 이 문서는 후순위 수익화 기능들을 정리한 것입니다.
> Phase 4 심리 케어 기능 완료 후 진행 예정입니다.

---

## 마케팅 전략

### 포지셔닝: "타임캡슐" 프레임

DearMe는 단순한 일기장이 아닌 **"디지털 타임캡슐"**로 포지셔닝합니다.

```
❌ 부정적 프레임 (피해야 할 것)
"죽기 전에 남기는 기록" / "죽은 후에도 대화" / "디지털 영생"

✅ 긍정적 프레임 (사용해야 할 것)
"미래의 나에게 보내는 편지" / "10년 후 나와의 대화" / "성장하는 나의 기록"
```

### 단계별 마케팅 메시지

| Phase | 시기 | 핵심 메시지 | 타겟 |
|-------|------|------------|------|
| **Phase 1** | 런칭 | "오늘의 나를 기록하고, 내일의 나와 대화하세요" | 자기 성찰 추구자 |
| **Phase 2** | 성장 | "친구의 마음을 이해하는 새로운 방법" | 관계 깊이 추구자 |
| **Phase 3** | 확장 | "아이에게 물려줄 나만의 이야기" | 디지털 레거시 관심자 |

### 타겟별 획득 채널

| 세그먼트 | 추천 채널 | 접근 방식 |
|----------|----------|----------|
| **자기 성찰 추구자** | 명상 앱, MBTI 커뮤니티, 자기계발 유튜브 | "나를 더 깊이 이해하는 도구" |
| **관계 깊이 추구자** | 연애/우정 커뮤니티, 인스타그램 | "친구와 더 가까워지는 방법" |
| **디지털 레거시** | 육아 커뮤니티, 가족 앨범 서비스 | "자녀에게 남기는 나의 이야기" |

### 경쟁 환경 분석

#### 직접 경쟁자

| 서비스 | 유형 | DearMe와의 차별점 |
|--------|------|-----------------|
| Replika | AI 친구 챗봇 | 페르소나가 **내 일기 기반**으로 생성 |
| Daylio | 감정 일기 앱 | **AI 페르소나**와 대화 가능 |
| Journey | 저널링 앱 | **친구 페르소나**와 소통 가능 |

#### 디지털 레거시 시장

| 서비스 | 유형 | 특징 |
|--------|------|------|
| HereAfter AI | 음성 기반 추모 | 고인의 음성으로 대화 |
| StoryWorth | 질문 기반 책 | 가족 이야기를 책으로 출판 |
| Eternime | 디지털 아바타 | 서비스 종료됨 |

# DearMe 기능 기획 아이디어

> 페르소나 캐릭터와 성향 시스템을 더 재미있고 몰입감 있게 만들기 위한 기획 문서

---

## 1. 캐릭터 이미지 생성 시스템

### 1.1 현재 문제점

- 생성할 때마다 완전히 다른 스타일의 이미지가 나옴
- 여러 명의 인물이 등장하거나 구도가 일정하지 않음
- "내 페르소나"라는 정체성이 약함

### 1.2 해결 방향: 템플릿 + 스타일 프리셋

#### 고정 프레임 시스템

```
┌─────────────────────────────────┐
│                                 │
│      ┌───────────────┐          │
│      │               │          │
│      │   👤 캐릭터    │  ← 상반신/버스트샷
│      │   (단일 인물)  │  ← 정면 또는 3/4 측면
│      │               │          │
│      └───────────────┘          │
│                                 │
│   [심플한 단색/그라데이션 배경]   │
│                                 │
└─────────────────────────────────┘
```

**고정 요소 (변하지 않음)**
- 구도: 상반신 (bust shot)
- 인물 수: 반드시 1명
- 시선: 카메라를 향함
- 배경: 단순한 색상 또는 부드러운 그라데이션

**변동 요소 (페르소나 특성 반영)**
- 표정 (밝은/차분한/신비로운)
- 헤어스타일 & 색상
- 의상 스타일 & 색상
- 악세서리 (안경, 귀걸이, 모자 등)
- 분위기 조명

### 1.3 스타일 프리셋 시스템

사용자가 페르소나 첫 생성 시 **아트 스타일을 선택**합니다.
이후 재생성/업데이트 시에도 동일한 스타일이 유지되어 캐릭터 정체성이 확립됩니다.

#### 스타일 옵션

| 스타일 | 미리보기 느낌 | 어울리는 성향 | 설명 |
|--------|-------------|--------------|------|
| 🎨 **따뜻한 수채화** | 부드러운 파스텔톤 | 감성적, 따뜻한 | 물감이 번지는 듯한 부드러운 터치 |
| ✏️ **미니멀 일러스트** | 깔끔한 선화 | 논리적, 심플한 | 군더더기 없는 세련된 라인 |
| 👾 **픽셀아트** | 레트로 도트 | 유쾌한, 개성있는 | 게임 캐릭터 같은 귀여운 느낌 |
| ✨ **애니메이션** | 일본 애니 스타일 | 밝은, 활발한 | 친근하고 생동감 있는 표현 |
| 🌙 **무드 일러스트** | 감성적 명암 | 깊이있는, 사색적 | 빛과 그림자가 아름다운 분위기 |
| 🖼️ **유화풍** | 클래식한 질감 | 성숙한, 고전적 | 명화 속 인물 같은 격조 |

#### 예시: 같은 페르소나, 다른 스타일

```
페르소나: "따뜻하고 공감 능력이 뛰어난 감성파"

[따뜻한 수채화]          [미니멀 일러스트]        [애니메이션]
┌──────────────┐      ┌──────────────┐      ┌──────────────┐
│  부드러운      │      │  깔끔한        │      │  밝고 선명한   │
│  파스텔 톤     │      │  단순한 선     │      │  색감과        │
│  따뜻한 미소   │      │  미니멀 표현   │      │  큰 눈         │
│  번지는 배경   │      │  화이트 배경   │      │  귀여운 표정   │
└──────────────┘      └──────────────┘      └──────────────┘
```

### 1.4 페르소나 특성 → 외형 매핑

일기 분석으로 추출된 성격 특성을 시각적 요소로 변환합니다.

#### 성격 → 색상 팔레트

| 주요 성향 | 메인 컬러 | 서브 컬러 | 분위기 |
|----------|----------|----------|--------|
| 따뜻함/공감 | 코랄, 피치 | 크림, 베이지 | 포근한 |
| 열정/에너지 | 오렌지, 레드 | 골드, 옐로우 | 활기찬 |
| 차분함/지성 | 네이비, 딥블루 | 그레이, 실버 | 신뢰감 |
| 창의/자유 | 퍼플, 라벤더 | 민트, 스카이 | 몽환적 |
| 자연/힐링 | 그린, 올리브 | 브라운, 우드 | 편안한 |
| 신비/깊이 | 다크퍼플, 버건디 | 골드, 블랙 | 미스터리 |

#### 성격 → 표정/포즈

| 성향 키워드 | 표정 | 포즈/제스처 |
|------------|------|------------|
| 밝음, 긍정적 | 환한 미소, 살짝 벌린 입 | 고개 살짝 갸웃 |
| 차분함, 사려깊음 | 은은한 미소, 부드러운 눈 | 정면 응시 |
| 호기심, 창의적 | 반짝이는 눈, 살짝 올라간 입꼬리 | 턱 괴기 |
| 내성적, 감성적 | 수줍은 미소, 내려간 시선 | 손으로 책/컵 들기 |
| 당당함, 리더십 | 자신감 있는 표정 | 팔짱 또는 손 턱 |
| 따뜻함, 배려 | 다정한 눈웃음 | 양손 모으기 |

#### 예시: 일기 분석 → 캐릭터 생성

```
📔 일기 분석 결과:
- 주요 성향: 감성적(85%), 내향적(70%), 창의적(65%)
- 자주 쓰는 표현: "오늘 하늘이 예뻤다", "음악을 들으며..."
- 감정 톤: 차분함, 사색적

🎨 생성 프롬프트:
"single person portrait, bust shot, soft watercolor style,
 young person with gentle dreamy expression,
 lavender and cream color palette,
 slightly tilted head, holding a small book,
 soft gradient background in pastel purple,
 warm lighting, peaceful atmosphere"

🖼️ 결과물:
┌─────────────────────────┐
│   라벤더 톤 그라데이션    │
│                         │
│      📖 책을 든          │
│      몽환적 표정의        │
│      캐릭터              │
│                         │
│   부드러운 수채화 느낌    │
└─────────────────────────┘
```

자동 생성 후 사용자가 미세 조정할 수 있는 옵션을 제공합니다.

```

├── 헤어 스타일: 숏컷 / 미디엄 / 롱 / 묶음머리

[분위기]

└── 배경색: 🟠 🟡 🟢 🔵 🟣 ⚪
```

### 1.6 시즌/기분별 변화 (재미 요소)

페르소나가 계절이나 최근 일기 기분에 따라 **의상/소품이 변화**합니다.

```
🌸 봄 (3-5월)
└── 파스텔 의상, 꽃 악세서리, 벚꽃 배경

☀️ 여름 (6-8월)
└── 밝은 색상, 선글라스, 시원한 배경

🍂 가을 (9-11월)
└── 니트/가디건, 따뜻한 톤, 단풍 배경

❄️ 겨울 (12-2월)
└── 따뜻한 옷, 머플러, 눈 배경
```

```
최근 7일 기분 분석:

😊 기쁨 70% → 밝은 표정, 화사한 색상
😢 슬픔 50% → 차분한 표정, 부드러운 톤
😤 분노 30% → 강렬한 눈빛, 진한 색상
😌 평온 80% → 온화한 미소, 파스텔 톤
```

## 2. 성향 시스템 (5대 축 + 게이미피케이션)

### 2.1 현재 문제점

- 키워드가 단순 나열되어 성장 재미가 없음
- 시간에 따른 변화를 체감하기 어려움
- "그래서 나는 어떤 사람인데?"에 대한 답이 모호함

### 2.2 해결: 5대 성향 축 시스템

일기에서 추출한 수많은 키워드를 **5가지 대분류**로 정리합니다.

```
              감성력 (Sensitivity)
                    ★★★★☆
                      ╱╲
                     ╱  ╲
                    ╱    ╲
   창의력 ─────────●─────────── 사교력
  (Creativity)    ╱ ╲          (Sociability)
   ★★★☆☆       ╱   ╲         ★★☆☆☆
                ╱     ╲
               ╱       ╲
   분석력 ◆─────────────────◆ 공감력
  (Analysis)              (Empathy)
   ★★★★☆                ★★★★★

        [나의 성향 레이더 차트]
```

#### 5대 성향 상세 정의

| 축 | 아이콘 | 정의 | 하위 키워드 예시 |
|----|--------|------|-----------------|
| **감성력** | 🎭 | 정서적 깊이와 예술적 감수성 | 낭만적, 센티멘탈, 예술적, 시적, 몽환적 |
| **창의력** | 💡 | 독창성과 상상력, 새로움 추구 | 창의적, 실험적, 자유로운, 호기심, 혁신적 |
| **사교력** | 🎉 | 대인관계 능력과 외향성 | 친화력, 유머, 리더십, 활발함, 사교적 |
| **분석력** | 🔍 | 논리적 사고와 체계성 | 계획적, 체계적, 신중한, 논리적, 현실적 |
| **공감력** | 💝 | 타인 이해와 정서적 교감 | 배려심, 경청, 따뜻함, 이해심, 섬세함 |

### 2.3 레벨 시스템

각 성향 축은 **레벨 1~10**까지 성장합니다.

#### 경험치 획득

```
📔 오늘의 일기 분석 완료!

┌─────────────────────────────────────┐
│  🎭 감성력    +15 EXP               │
│  ━━━━━━━━━━━━━━━━░░░░ Lv.4 (67%)    │
│                                     │
│  💝 공감력    +25 EXP  ⬆️ COMBO!     │
│  ━━━━━━━━━━━━━━━━━━░░ Lv.5 (89%)    │
│                                     │
│  💡 창의력    +10 EXP               │
│  ━━━━━━━━━░░░░░░░░░░░ Lv.3 (42%)    │
└─────────────────────────────────────┘

💬 오늘 일기에서 발견한 것:
   "친구의 고민을 들어줬다" → 공감력 +20
   "노을이 예뻐서 사진 찍었다" → 감성력 +15
   "새로운 카페를 발견했다" → 창의력 +10
```

### 2.4 콤보 & 스트릭 시스템

연속으로 특정 성향이 감지되면 **콤보 보너스**가 적용됩니다.

```
🔥 공감력 5일 연속! COMBO x1.5

Day 1: 친구 위로 → +10
Day 2: 가족 감사 → +12
Day 3: 동료 칭찬 → +15
Day 4: 봉사활동 → +18
Day 5: 이웃 도움 → +22 (x1.5 = +33!)

━━━━━━━━━━━━━━━━━━━━━
🎯 7일 연속 달성 시: [공감 마스터] 칭호!
━━━━━━━━━━━━━━━━━━━━━
```

### 2.5 칭호 & 뱃지 시스템

성장과 활동에 따라 다양한 칭호를 획득합니다.

#### 레벨 기반 칭호

| 성향 | Lv.3 | Lv.5 | Lv.7 | Lv.10 (MAX) |
|------|------|------|------|-------------|
| 🎭 감성력 | 감성 새싹 | 감성 탐험가 | 감성 예술가 | 🌙 **밤의 시인** |
| 💡 창의력 | 아이디어 씨앗 | 발상의 전환 | 창조의 불꽃 | ✨ **무한 상상가** |
| 🎉 사교력 | 친구의 친구 | 분위기 메이커 | 인싸 of 인싸 | 🎪 **파티 마스터** |
| 🔍 분석력 | 관찰자 | 전략가 | 논리의 달인 | 🧠 **마스터 플래너** |
| 💝 공감력 | 따뜻한 마음 | 경청의 귀 | 마음의 치유사 | 💫 **영혼의 동반자** |

#### 특별 칭호 (도전 과제)

```
🏆 특별 칭호 컬렉션

[일상 기록가]
└── 30일 연속 일기 작성
    "꾸준함이 만드는 기적"

[균형 잡힌 사람]
└── 모든 성향 Lv.5 이상 달성
    "어떤 상황에서도 빛나는 당신"

[감정의 무지개]
└── 한 달간 7가지 이상 감정 기록
    "다채로운 감정의 소유자"

[새벽형 인간]
└── 새벽 5시 이전 일기 10회 작성
    "고요한 시간의 친구"

[계절의 증인]
└── 4계절 모두 일기 작성
    "시간의 흐름을 담는 사람"

[단짝 친구]
└── 한 친구 페르소나와 100회 대화
    "깊어지는 우정의 증표"
```

         혹시 다른 사람들 일까지 떠안고 있는 거 아냐?

         네 감성력 🎭 레벨을 보면
         혼자만의 시간이 필요한 타입인 것 같은데,
         이번 주말엔 조용히 쉬는 건 어때?

나: 새로운 프로젝트 시작하고 싶어
